# AUX

## A web application for friends to communally queue spotify songs

## How to run:

	git clone "repo link"
	cd aux
	npm install
	npm start

## How we are going to develop:

Never push to master.  Branch names should be in the form of: 
	
	dev's initials-change worked on

	my-fixnavbar
